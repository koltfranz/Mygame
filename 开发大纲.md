## 第零部分：AI 编码绝对红线（前置系统指令）

在阅读后续任何技术需求之前，AI 必须在系统提示层锁死以下规则，违背任何一条即视为致命 Error：

1. **本体论红线**：绝对禁止将“价值”或“使用价值”声明为物品的静态属性（严禁 `item.value = x` 或 `item.use_value = y`）。使用价值是动态判定事件，价值是事后结算的幽灵比例。

2. **行为学红线**：绝对禁止在 Agent 的 `step()` 中写入任何基于“贪婪、效用最大化、避险”等抽象人性的 if-else 逻辑。Agent 行为必须且只能从 `SocialRelationGraph` 的连边推导。

3. **术语红线**：代码注释、变量命名、日志输出必须严格使用第七部分附录 A 的术语表。触及附录 B 的禁语表任何一词，必须立即重构。

4. **机制红线**：机器绝对不能创造新价值（只能转移 $c$）。工资绝对不能等于劳动的报酬（只能是劳动力价值的掩蔽形式）。危机绝对不能由主观需求不足引起（只能由 $p' = s/(c+v)$ 下降或再生产比例失衡引起）。

5. **测试红线**：任何新功能或修改的 Pull Request 必须附带至少一个位于 `tests/test_marxist_laws.py` 中的测试用例，用以验证本次变更未违反上述四条红线。

6. **先验性红线**：绝对禁止预设复杂劳动的倍加系数。绝对禁止预设固定技能要求表。技能要求必须由技术知识、工具质量和劳动者技能分布动态计算。

## 第二部分：Mesa 模型与理论核心设计

### 2.1 底层本体论：物品四态机与生产资料磨损

任何物质团块在内存中必须处于以下四种状态之一：

1. **`State_Useless` (无用物)**：物理上无法满足当前人类需求。
2. **`State_Pure_Use_Value` (纯使用价值)**：自然界直接提供、无劳动介入即满足需求之物。
3. **`State_Product` (产品)**：**凝结了人类具体劳动的有用物**。具有 `physical_props` 和 `individual_labor_embodied`。
4. **`State_Commodity` (商品)**：产品跨越社会边界的跃迁态。包含 `use_value_props`、`individual_labor_embodied`、`exchange_status="Pending"`。

**物品职能标签**：

| 标签 | 含义 | 示例 |
|:---|:---|:---|
| `edible` | 可食用 | 谷物、肉 |
| `seed` | 可作为种子 | 谷物 |
| `tool` | 可作为工具 | 石器、机器 |
| `raw_material` | 可作为原料 | 铁矿石、木材 |
| `luxury` | 奢侈品属性 | 金、香料 |
| `money_commodity` | 货币商品属性 | 金 |
| `weapon` | 武器 | 火枪 |
| `location_change` | 运输服务 | 追加价值的特殊商品 |

**部类动态判定**：`determine_sector(matter, context)` 根据交易或使用上下文返回 `SECTOR_I`（生产资料）、`SECTOR_II`（消费资料）、`SECTOR_TRANSPORT`（运输）或 `SECTOR_MONEY`（货币商品）。

**使用价值丧失与价值丧失**：

- `use_value_loss_check()`：物品因时间、使用或环境失去有用性，状态降为 `State_Useless`。
- 若商品处于 `Pending` 状态且发生使用价值丧失，触发 `unrealized_commodity_value_loss` 事件，SNLT计算器将其移出待结算池，其凝结的私人劳动不被社会承认为价值。

**生产资料磨损与价值转移**：

- **有形磨损（物理磨损）**：由使用和自然力引起的使用价值丧失，价值按磨损比例转移。
- **无形磨损（精神磨损）**：因该部门SNLT下降导致的原有资本贬值。
- **价值转移量** = 原始价值 × (1 - 剩余使用价值比例 × 精神磨损系数) - 累计已转移价值。
- 若剩余使用价值比例归零或精神磨损使其价值归零，生产资料报废，触发 `means_of_production_scrapped` 事件。

- 地图尺寸随Agent数量自适应。

物品按历史阶段解锁，需求分L1（生理生存）、L2（基本再生产）、L3（社会必要需求）、L4（奢侈需求），由阶级位置强制决定。

### 2.11 上层建筑模块

- **国家机器**：镇压、财政、行政能力。
- **法律体系**：所有权法、契约法、人身法、劳动法。立法由阶级斗争触发，司法存在阶级偏见。
- **意识形态**：霸权强度影响工人意识，合法性危机触发革命。
- **政体形式**：由阶级力量对比决定。

### 2.12 国际贸易与外部市场

- `ForeignMarket` 抽象外部市场，国际价值取国内外SNLT较低者。
- 贸易影响部类平衡：$I(v+m) + Export_I = II(c) + Import_c + Import_v$。

### 2.13 殖民与原始积累

`Colony` 类，榨取模式包括贵金属掠夺、种植园奴隶制、移民殖民地、商品倾销。净收益为负且抵抗烈度高时触发独立事件。

### 2.14 军事与国家暴力

军队是镇压性国家机器核心，军工属第I部类。战争消耗剩余价值，也可作为危机外部出口毁灭过剩资本。

### 2.15 人口动态与产业后备军

生育率和死亡率由阶级、收入、城市化、教育、医疗等社会经济变量决定。人口转型参照历史西欧模式。产业后备军规模影响工资和阶级斗争。

### 5.1 物品状态跃迁与使用价值丧失

```python
class Matter:
    def __init__(self):
        self.state = State_Useless
        self.physical_props = {}
        self.individual_labor_embodied = 0.0
        self.exchange_status = None
        self.use_value_lifespan = 100
        # 生产资料特有属性
        if self.physical_props.get("means_of_production"):
            self.physical_wear_rate = 0.01
            self.idle_wear_rate = 0.001
            self.remaining_use_value_ratio = 1.0
            self.moral_depreciation_factor = 1.0
            self.original_value = 0.0
            self.transferred_value_accumulated = 0.0

    def check_use_value_loss(self):
        self.use_value_lifespan -= 1
        if self.use_value_lifespan <= 0:
            if self.state == State_Commodity and self.exchange_status == "Pending":
                EventBus.emit("unrealized_commodity_value_loss", self)
            self.state = State_Useless
            self.individual_labor_embodied = 0.0
```

### 5.2 生产资料磨损与价值转移

```python
def apply_wear_and_transfer_value(machine, production_quantity):
    physical_wear = machine.physical_wear_rate * production_quantity
    idle_wear = machine.idle_wear_rate
    total_wear = physical_wear + idle_wear

    current_snlt = SNLTCalculator.get_snlt(machine.sector)
    original_snlt = machine.original_snlt_at_production
    moral_factor = current_snlt / original_snlt
    machine.moral_depreciation_factor = moral_factor

    machine.remaining_use_value_ratio -= total_wear

    effective_wear = 1 - (machine.remaining_use_value_ratio * moral_factor)
    value_transferred = machine.original_value * effective_wear - machine.transferred_value_accumulated
    machine.transferred_value_accumulated += value_transferred

    if machine.remaining_use_value_ratio <= 0 or moral_factor <= 0.01:
        machine.state = State_Useless
        EventBus.emit("means_of_production_scrapped", machine)

    return value_transferred
```

```python
    foreign_snlt = foreign_market.get_snlt(domestic_commodity.commodity_type)
    domestic_snlt = domestic_commodity.snlt
    realized_value = min(domestic_snlt, foreign_snlt)
    if domestic_snlt < foreign_snlt:
        super_profit = foreign_snlt - domestic_snlt
    return realized_value
```
